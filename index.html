Sobre o Projeto

O FormGestor é um sistema criado para que representantes de turma possam registrar de forma organizada e intuitiva as aulas ministradas, com dados como: professor, unidade curricular, alunos presentes e horários. A coordenação tem total controle para cadastrar/remover turmas, gerenciar alunos e exportar planilhas.

Funcionalidades

📘 Cadastrar / Remover turmas
🧑‍🎓 Adicionar / Remover alunos
🗓️ Registrar aulas por data
📎 Exportar planilhas por turma

Diferenciais

  • Interface clara para representantes
  • Painel exclusivo para coordenação
  • Design responsivo e acessível
  • Automação na geração de relatórios
  • Separação de planilhas por turma
  • Pronto para uso em sala de aula
